# PVC

Pipe between processes as easily as in the shell

## Check it out

    # run the specs
    bundle exec rspec spec

## Install it as a RubyGem

This code is packaged as a Gem. If you like, you can build and install it by running:

    gem build pvc.gemspec
    gem install pvc-*.gem
